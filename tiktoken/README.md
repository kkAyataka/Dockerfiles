# tiktoken

tiktoken[^1] を使ってトークン数を計算して表示する。

* 文字列を指定すると、その文字列のトークン数を返す
* ファイルパスを指定すると、そのファイルのトークン数を返す
* ディレクトリパスを指定すると、ディレクトリを再帰的に調べて、すべてのファイルのトークン数と合計値を表示する

`o200k_base` のエンコーダーがあらかじめキャッシュされており、オフラインで動作する。

以下のように実行すると、現在のディレクトリ下のファイルのトークン数が表示される。

```console
docker run --rm -v ./:/work tiktoken .
```

[^1]: <https://github.com/openai/tiktoken>
